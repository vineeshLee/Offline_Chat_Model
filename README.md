# Offline Chat Model

## Project Overview
The Offline Chat Model is designed to facilitate seamless communication using advanced natural language processing techniques. It allows users to engage in real-time conversations without relying on an internet connection.

## Features
- Real-time chat capabilities
- Integration with Hugging Face models for better conversational understanding
- Easy setup and usage
- Ability to run in offline mode

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/vineeshLee/Offline_Chat_Model.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Offline_Chat_Model
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## How to Run
To run the Offline Chat Model, execute the following command:
```bash
python chat_model.py
```

## How to Test
You can test the offline chat model using a Hugging Face model integration. Make sure you have the model downloaded and accessible in your environment. Use the following command to test:
```bash
python test_model.py --model your_model_name
```
